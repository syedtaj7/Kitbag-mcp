# utility.unit_convert

Convert values between physical or data units (length, mass, temperature, data storage).

## Usage

Provide `value`, `type`, `from`, and `to`.

### Input Schema

- `value` (number, required): Source value.
- `type` (string, required): 'length', 'mass', 'temperature', 'data'.
- `from` (string, required): Source unit.
- `to` (string, required): Target unit.
