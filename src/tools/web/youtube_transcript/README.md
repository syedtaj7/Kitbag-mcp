# web.youtube_transcript

Extract text transcripts and captions with timestamps from a YouTube video URL.

## Usage

Provide the YouTube video `url` and optional `languageCode`.

### Input Schema

- `url` (string, required): The URL of the YouTube video.
- `languageCode` (string, optional): Preferred language code (e.g. 'en', 'es'). Defaults to 'en'.
